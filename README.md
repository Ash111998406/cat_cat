# 🐱 Kitten Rescue

A mobile & desktop friendly browser game. Tap the mysterious box, then drag the runaway kittens into the cage before time runs out. Works with touch screens and mouse.

## How to publish this with GitHub Pages

1. Create a new repository on GitHub (or use an existing one).
2. Upload `index.html` to the repository (root folder, or a `/docs` folder — just note which one you pick).
3. Go to your repo → **Settings** → **Pages**.
4. Under "Build and deployment", set **Source** to "Deploy from a branch".
5. Pick the branch (usually `main`) and the folder (`/root` or `/docs`, matching where you put `index.html`).
6. Click **Save**. GitHub will give you a link like:
   `https://<your-username>.github.io/<repo-name>/`
7. Open that link on your phone or computer — the game will load directly, no installs needed.

That's it — every time you push changes to `index.html`, the live page updates automatically within a minute or two.
